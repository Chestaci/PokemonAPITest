Кейс 1. Проверка, что у покемона rattata, в отличие от покемона pidgeotto, меньше вес и есть умение
(ability) побег (run-away)

Параметры тест кейса: {name}

1. rattata, pidgeotto

Шаги:

1. Отправить GET запрос https://pokeapi.co/api/v2/pokemon/{name}/
2. Проверить код состояния
3. Проверить тело ответа от сервера
4. Проверить, что у покемона rattata вес меньше, чем у pidgeotto
5. Проверить, что у покемона rattata есть умение (ability) побег (run-away)
6. Проверить, что у покемона pidgeotto нет умения (ability) побег (run-away)


Ожидаемый результат:

1. Запрос успешно отправлен на сервер
2. Код состояния: HTTP Status 200 OK
3. Тело ответа в формате JSON
4. У покемона rattata меньше вес, чем у от покемона pidgeotto
5. У покемона rattata есть умение (ability) побег (run-away)
6. У покемона pidgeotto нет умения (ability) побег (run-away)

Параметры тест кейса: {limit}

1. 30
2. 50

Шаги:

1. Отправить GET запрос https://pokeapi.co/api/v2/pokemon/?limit={limit}
2. Проверить код состояния
3. Проверить тело ответа от сервера
4. Проверить ограничение списка {limit}
5. Проверить наличие имен у покемонов из ограниченного списка


Ожидаемый результат:

1. Запрос успешно отправлен на сервер
2. Код состояния: HTTP Status 200 OK
3. Тело ответа в формате JSON
4. Список покемонов ограничен и равен = {limit}
5. У каждого покемона в списке есть имя
